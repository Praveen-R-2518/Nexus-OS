# 3D model attribution

## `macbook-m5.glb`

- **Source:** ["MacBook Pro 14-inch M5" by Apple User (@apple-user)](https://sketchfab.com/3d-models/macbook-pro-14-inch-m5-652a992f4f244122ae251f9cbb81da1e) on Sketchfab
- **License:** Sketchfab Standard (free download; commercial web use permitted)
- **Modifications:** textures resized to 1024px + WebP, Draco mesh compression
  (via `@gltf-transform/cli`); materials re-tinted Space Black and the display
  material replaced at runtime with the Nexus OS dashboard canvas texture.
- **Usage:** MacBook Pro marketing hero on the Nexus OS landing page
  (`components/landing/MacbookModel.tsx`)
